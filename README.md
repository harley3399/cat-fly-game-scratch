# Cat Fly Game

A fun flying cat game built with [Scratch 3.0](https://scratch.mit.edu).

## How to Play

1. Download the file `cat fly-harley3399.sb3`
2. Go to [scratch.mit.edu](https://scratch.mit.edu)
3. Click **File** → **Load from your computer**
4. Select the `.sb3` file
5. Press the **Green Flag** to start
6. Control the cat to fly and avoid obstacles!

## Features

- Cute cat flying mechanic
- Obstacle dodging gameplay
- Built entirely in Scratch visual programming
