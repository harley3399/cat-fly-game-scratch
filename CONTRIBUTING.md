# Contributing to Cat Fly Game

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** this repository
2. **Download** the `.sb3` file and open it in [Scratch](https://scratch.mit.edu)
3. Make your changes or improvements
4. **Export** the updated `.sb3` file
5. Submit a **Pull Request** with your changes

## Ideas for Contributions

- New obstacles or enemies
- Sound effects and music
- Better backgrounds or animations
- Score system improvements
- New levels or difficulty settings
- Bug fixes

## Reporting Issues

If you find a bug or have a suggestion, please open an [Issue](../../issues) and describe:
- What happened
- What you expected to happen
- Steps to reproduce

## Code of Conduct

Be kind and respectful to everyone in this project.
